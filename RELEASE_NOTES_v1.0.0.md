# 边框实验室 v1.0.0

## 更新内容

- 新增 4 套模板：
  - 横屏白框
  - 竖屏白框
  - 横屏壹印
  - 竖屏壹印
- 新增右侧预览滚轮缩放与拖拽查看
- 新增导出最长边限制选项
- 新增圆角 PNG 导出
- 优化模板缩略图展示
- 优化壹印模板的高斯模糊底板与阴影表现
- 支持在信息栏追加 EXIF 焦段

## 已知说明

- JPG / JPEG 的 EXIF 读取最完整
- PNG / WebP 可以导入，但不一定带有完整拍摄信息
- 字体效果依赖本机是否安装对应字体

## 建议发布标题

`边框实验室 v1.0.0`

## 建议 Release 简述

首个可发布版本，包含白框和壹印两大类模板、圆角导出、最长边限制、实时预览缩放拖拽，以及更完整的 EXIF 信息展示能力。

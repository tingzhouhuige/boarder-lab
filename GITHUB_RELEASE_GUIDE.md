# GitHub Release 发布说明

## 建议上传内容

- `index.html`
- `styles.css`
- `app.js`
- `assets/`
- `README.md`
- `RELEASE_NOTES_v1.0.0.md`

## 推荐发布步骤

1. 把项目推送到 GitHub 仓库
2. 在仓库中创建 Tag：`v1.0.0`
3. 打开仓库 `Releases`
4. 点击 `Draft a new release`
5. 选择 `v1.0.0`
6. 标题填写：`边框实验室 v1.0.0`
7. 将 `RELEASE_NOTES_v1.0.0.md` 中内容粘贴到说明区
8. 上传发布压缩包
9. 点击发布

## 压缩包建议命名

`border-lab-v1.0.0.zip`

## 适合放入压缩包的文件

```text
index.html
styles.css
app.js
assets/
README.md
RELEASE_NOTES_v1.0.0.md
```
